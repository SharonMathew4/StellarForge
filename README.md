
# StellarForge

- build command:<br>
```bash
mkdir out && cd out && cmake .. && make -j12 && cd - && ./Project/ParticleSim/ParticleSim
```

- mouse control:
	- Left click: attract particles<br>
	- Rigth click: repel particles<br>
	- Middle click: move camera<br>
	- Scroll: zoom in/out
- keyboard control:
	- K: set all particle velocity to 0
